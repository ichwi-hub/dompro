# ДомПро

Локальный прототип веб-приложения для отслеживания строительства дома: этапы, прогресс и бюджет.

## Запуск

Откройте `index.html` в браузере или выполните:

```powershell
cd C:\projects\dompro
start index.html
```

## Структура

```
dompro/
├── index.html      # главная страница
├── css/style.css   # стили
└── js/app.js       # логика
```
