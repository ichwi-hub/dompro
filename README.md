# DomPro (ДомПро)

Премиальный вертикальный маркетплейс экспертов: юристы, бухгалтеры, маркетологи, дизайнеры и другие специалисты.

## Что это

DomPro — не строительная компания, а платформа, где:
- **эксперты** создают профиль, публикуют услуги и получают заказы;
- **клиенты** находят проверенных специалистов и оформляют безопасные сделки.

## Запуск

```powershell
cd C:\projects\dompro
start index.html
```
