# Чеклист онбординга эксперта DomPro

Пошаговое руководство для первого живого пользователя (например, юрист Валерия).

## Перед началом

- Сайт открыт: http://127.0.0.1:5500/frontend/login.html (или production URL)
- API работает: `/health` возвращает `healthy`
- На балансе для отклика нужно минимум **150 ₽**

---

## Шаг 1. Регистрация

1. Откройте **Регистрация эксперта** (`register-expert.html`)
2. Укажите email, телефон, пароль
3. После входа откроется **Expert Workspace** (`workspace.html#feed`)

**Частые проблемы:**
- `Connection Failed` — не запущен API или фронтенд (см. `backend/README.md`)
- Email уже занят — используйте другой или войдите через login

---

## Шаг 2. Профиль

1. Перейдите в **Кабинет эксперта** (`expert-dashboard.html`) или заполните профиль из уведомления
2. Нажмите **Редактировать профиль**
3. Заполните: ФИО, специализация, стаж, описание, образование (опц.), фото URL (опц.)
4. Сохраните

Без заполненного профиля верификация недоступна.

---

## Шаг 3. Верификация

1. В кабинете → блок **Верификация** → **Загрузить документы**
2. Приложите: ИНН, диплом, статус самозанятого/ИП
3. Дождитесь статуса `verification_pending`

Администратор одобряет через API:
- `GET /api/v1/admin/verifications`
- `PUT /api/v1/admin/verifications/{id}/approve`

После одобрения статус → `verified`, отклики доступны.

---

## Шаг 4. Кошелёк

1. В кабинете → **Пополнить** (заглушка, мгновенное зачисление)
2. Убедитесь, что баланс ≥ 150 ₽

---

## Шаг 5. Заказы и отклики

1. **Лента** в Workspace (`workspace.html#feed`) — новые заказы и активная работа
2. Или **Заказы** (`orders.html`) — все открытые заказы
3. Нажмите **Откликнуться**, напишите сообщение → списание 150 ₽

---

## Шаг 6. После принятия отклика

1. Клиент принимает отклик → заказ `in_progress`
2. **Мои клиенты** (`workspace.html#clients`) — появится заказчик
3. **Договоры** (`workspace.html#contracts`) или страница заказа → **Создать договор** → скачать PDF

---

## Шаг 7. Поддержка

При проблемах:
- Проверьте `/health` и консоль браузера (F12)
- Убедитесь, что верификация `verified`
- Напишите администратору платформы
